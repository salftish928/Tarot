# 塔罗牌占卜应用

## 文件结构

```
tarot/
├── index.html              # 主页面
├── style.css               # 样式文件
├── script.js              # 脚本文件
├── README.md              # 本说明文件
└── images/                # 图片文件夹
    └── tarot_cards_compressed/  # 压缩塔罗牌图片文件夹
        ├── 0愚者.png
        ├── 1魔术师.png
        ├── 2女祭司.png
        ├── ... (共78张图片)
```

## 功能说明

1. **抽牌功能**：支持连续抽取塔罗牌，直到78张全部抽完
2. **正逆位**：每张牌有50%概率出现正位或逆位，逆位卡牌图片会自动旋转180度
3. **主题切换**：支持四种主题（黑金、蓝金、红金、绿金）
4. **AI解读**：集成通义API进行AI解牌
5. **卡牌交互**：卡牌只能从背面翻转到正面，翻转后点击可查看解牌内容

## 使用方法

### 1. 运行应用

1. 使用本地服务器打开 `index.html`
2. 或使用Python运行：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器中访问 `http://localhost:8000`

### 2. 配置API Key（可选）

如需使用AI解读功能，请在 `script.js` 中填写您的通义API Key：

```javascript
const TONGYI_API_KEY = "您的API Key";
```

## 注意事项

1. **所有原有功能保持不变**：抽牌、正逆位、用户输入问题等逻辑完全不变
2. **图片路径**：代码已修改为从 `images/tarot_cards_compressed/` 文件夹读取本地压缩图片
3. **卡牌翻转**：卡牌只能从背面翻转到正面，翻转后不能再翻回去，再次点击可查看解牌
4. **逆位效果**：逆位卡牌的图片会自动旋转180度，直观显示逆位效果
